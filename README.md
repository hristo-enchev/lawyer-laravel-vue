## Features

- Laravel 7
- Vue + VueRouter + Vuex
- Login, register, email verification and password reset
- Authentication with Sanctum 
- Bootstrap 4

## Installation

- Clone the project
- Edit `.env` and set your database connection details
- `php artisan migrate`
- `npm install`
- `npm run dev`
